Decentralized gaming 12ta

==============Environmental installation steps==============\n
1. Install Node.js v22.15.1 version\n
node --version\n
npm --version\n
(Hint version number, proving that you have successfully installed)\n
2. Load oracle dependencies\n
npm install\n
3. Enter the web desktop directory to install serve (for local client startup)\n
npm i serve\n
==============Startup steps==============\n
1. Activate the oracle\n
node index.js\n
2. Run in the web desktop directory\n
npx serve -p 3001\n
3. Launch the client and input in Chrome browser\n
http://localhost:3001/\n
4. Create a key on the login page\n
5. Write the address into the oracle\n
